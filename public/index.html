How to navigate through this gallery API

Through localhost:3000

  • HOME: http://localhost:3000/
  • QUOTES: api/quotes
  • ID: api/quotes/:id
  • 404: http://localhost:3000/a

Through Heroku App

  • HOME: https://inspire-cpnt262-a5.herokuapp.com/
  • QUOTES: api/quotes/
  • ID: api/quotes/ + number
  • 404: https://inspire-cpnt262-a5.herokuapp.com/a
